# Portfolio Website – Assignment 3

## Project Description
This project is a personal portfolio website designed to showcase projects, demonstrate front-end development skills, and provide an interactive user experience.

The application includes dynamic content, advanced logic, and state management features built using HTML, CSS, and JavaScript.

---

## Features
- Responsive portfolio design
- Project filtering, searching, and sorting
- Developer Advice API integration
- Contact form with validation
- Light/Dark theme with saved preference
- Personalized welcome message
- Session timer tracking user time on site

---

## API Integration
The application uses the Advice Slip API to fetch and display live developer advice dynamically.

Users can click "Get New Advice" to retrieve new content. If the API request fails, a user-friendly error message is displayed.

---

## How to Run Locally
1. Download or clone the repository
2. Open `index.html` in any web browser
3. No installation or additional setup is required

---

## How to Use the Website
1. Use the navigation bar to move between sections (About, Projects, Advice, Contact).
2. In the Projects section:
    - Filter by category
    - Search by project name
    - Sort projects alphabetically
3. In the Developer Advice section:
    - Click **Get New Advice** to load new advice from the API
4. In the Personalized Welcome section:
    - Enter your name and click **Save Name**
    - The greeting will update and remain saved after refresh
5. In the Contact section:
    - Fill in all fields
    - Fix any validation errors
    - Submit the form
